# [4. Median of Two Sorted Arrays](https://leetcode.com/problems/median-of-two-sorted-arrays/#/description)

## 描述

有两个大小分别为m和n的已排序数组nums1和nums2。

找到两个排序数组的中位数。 要求程序的总时间复杂度应该是O(log(m + n))。

**Example 1:**

```
nums1 = [1, 3]
nums2 = [2]

The median is 2.0
```

**Example 2:**

```
nums1 = [1, 2]
nums2 = [3, 4]

The median is (2 + 3)/2 = 2.5
```
